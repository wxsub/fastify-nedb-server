## 可用脚本

在项目目录中，您可以运行：

### `npm run dev`

要以开发模式启动应用程序。\

打开 [http://localhost:3000](http://localhost:3000) 在浏览器中查看。

### `npm start`

要以生产模式启动应用程序。
